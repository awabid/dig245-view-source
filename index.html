DIG 245 View Source

Want to know how to view the source code of a website on Google Chrome?
Follow these steps:

  1. Right-click at any blank space on the web page
  2. To simply view the html code, click "View Page Source", and it will open a new window displaying the html code
  3. To use advanced developer tools, find and click "Inspect". This will open up a new inspection window on the right.
  4. This inspection window gives you access to Chrome's Dev Tools, to view/edit the website source code
  5. In the inspection window, you can view and edit HTML code in the "Elements" tab

What can you do with the Chrome Dev Tools?

  • Edit the CSS elements: observe the "Styles" tab in the bottom-right window. Here you can see the CSS styles applied to the website and manipulate them!
  • Use the Console: In the "Console" tab, you can type and execute commandline instructions and code, like Javascript.
  • View sources: In the "Sources" tab, you can view the file directory of the website, and observe where and how the different webpages and files are linked. Click on any file to view its contents. You can even view pictures!

Fun, right? In a sense, viewing the source code is like piercing the veil of web interfaces
